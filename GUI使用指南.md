# GUI使用指南

## 启动

双击 `START_GUI.bat` 或 `START_GUI_TEST.bat`

## 配置

1. **目标场数**: 输入要执行的总场数（如100）
2. **服务器路径**: 选择 `guandan_offline_v1006.exe`
3. **客户端脚本**: 输入4个客户端路径，逗号分隔
   - 前2个是你的AI（一队）
   - 后2个是对手AI（另一队）

## 示例配置

```
目标场数: 100
服务器: guandan_offline_v1006.exe
客户端: src/communication/Test1.py, src/communication/Test2.py, D:\对手\client.py, D:\对手\client.py
```

## 执行

1. 点击"开始执行"
2. 观察进度条和日志
3. 系统会自动重启完成100场（服务器单次限制3场）
4. 完成后显示最终战绩

## 常见问题

**只运行3场就停止？**
- 已修复，系统会自动重启继续执行

**GUI不显示状态？**
- 已修复，确保关闭旧版本重新启动

**找不到服务器文件？**
- 点击"浏览"按钮选择 `.exe` 文件
